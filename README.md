# 📚 JavaScript Flash Card Quiz Game

A beginner-friendly Flash Card Quiz Game built using **HTML, CSS, and JavaScript**. Users can answer JavaScript-based quiz questions, submit responses, flip flashcards to reveal answers, and view their final score after completing the quiz.

## ✨ Features

* 20 beginner-level JavaScript questions
* User input answer system
* Answer validation
* Interactive flashcard flip animation
* Correct/Wrong feedback
* Question counter
* Final score display
* Play Again option
* Minimal pastel UI design
* Smooth animations and hover effects
* Responsive layout

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎮 How to Use

1. Read the question displayed on the flashcard.
2. Enter your answer in the input box.
3. Click **Submit**.
4. See whether your answer is correct or wrong.
5. Tap the flashcard to flip and reveal the correct answer.
6. Click **Next Question** to continue.
7. After completing all questions, view your final score.
